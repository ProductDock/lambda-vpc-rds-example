# Lambda-Vpc-RDS configuration

Configure your Lambda function to access Amazon RDS database in VPC using AWS CDK and Golang.

## Prerequisites

* Golang
* Aws cdk
* AWS CLI credentials 

## Commands:

* Deploy: `cdk deploy --all`
* Destroy: `cdk destroy --all`
